# jchoxha.github.io

A directory of my live GitHub Pages sites, served at the root:
**https://jchoxha.github.io/**

It's a single self-contained `index.html` (no build step) with a dark, playful
look — a glowing animated gradient backdrop, frosted-glass cards, and a per-card
accent color that glows on hover. To add or edit a site, update the `SITES` array near the
bottom of `index.html` — each entry is one object (`emoji`, `title`, `desc`,
`url`, and an optional `color`; a palette is auto-assigned if you omit it).
Push to `main` and GitHub Pages redeploys automatically.

## Listed sites
- [str.rest](https://jchoxha.github.io/str-rest/) — digital guidebooks for short-term rental hosts
- [Chimera Cards](https://jchoxha.github.io/chimera_cards/)
- [HappyPrism](https://jchoxha.github.io/HappyPrism-Canvas/)
